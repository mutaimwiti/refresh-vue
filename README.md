# refresh-vue
A simple todo list application done in Vue.js

[Open App](https://mutaimwiti.github.io/refresh-vue/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
